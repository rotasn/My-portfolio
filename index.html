  • Marauder

    Description

    An automatic mapping tool for Interactive Fiction games.

    • Automatic mapping of the game's rooms and connections.
    • A multi-layer display system to display rooms on different floors
    • Quick travel to the desired room
    • Note taking for rooms
    • Automatic exploration (using an agent to build the map)
    • Save/Load maps for future playthroughs

    Built with Electron, using JavaScript and a bit of Python. Room detection uses rule-based filtering and NLP. Quick travel uses BFS. Auto-exploration uses DFS + backtracking. Maps are saved as JSON.
